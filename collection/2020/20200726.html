# 开源软件

  • PasteMe一个文本分享网站，不需要注册，会对输入的文本或代码生成 URL，支持密码和阅后即焚。
  • Passbolt一个开源的密码管理器，可以自己架设 Web 服务。
  • Rust 语言的中文版在线教程，通过可以运行的小例子介绍语法，从 Hello World 开始讲起。
  • Wiki.js架设个人 Wiki 的 Node.js 应用。
  • Easy-Monitor:Node.js 应用的性能监控工具，可以定位线上故障。代码开源，支持私有部署。
  • webtorrent-desktop一个 BT 下载的跨平台桌面客户端，可以一边下载一边观看视频。它使用 WebTorrent 协议，所以实际上是一个网页应用，用 Electron 包装成桌面软件。
